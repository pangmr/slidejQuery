# slidejQuery
轮播图
使用：
  HTML结构：
    <div class="slider">
        <ul class="slides">
            <li><img src="img/banner0.jpg"></li>
            <li><img src="img/banner1.jpg"></li>
            <li><img src="img/banner2.jpg"></li>
        </ul>
    </div>
1.引用文件：
    <link rel="stylesheet" href="css/slide.css">
    <script src="js/slide.jquery.js"></script>
    
2.调用方法：
    $('.slider').swiper();
